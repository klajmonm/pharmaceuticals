# Pharmaceuticals

Optimized geometries (.xyz), COSMO-files (.cosmo), and sigma-profiles (.sigma) of 16 pharmaceuticals:

+ COSMO-files calculated by Gaussian 16 at the BVP86/TZVP/cosmors level
+ sigma-profiles generated using the 'to_sigma.py' script (https://doi.org/10.1021/acs.jctc.9b01016) with --n 3 and --averaging Hsieh

plot_sigma_APIs.pdf shows sigma-profiles of all 16 pharmaceuticals.
